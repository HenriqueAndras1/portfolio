Sobre Henrique da Silva de Andradegit

Meu nome é Henrique, tenho 30 anos e moro na zona leste de São Paulo. Gosto de praticar esportes aos finais de semana, como futebol e andar de bicicleta, e atualmente estou aprendendo a tocar violão. Tenho 10 anos de experiência em varejo, sendo minha última experiência na Obramax, onde fui vendedor pleno por 4 anos. Nesse período, desenvolvi habilidades em trabalho em equipe, liderança, autonomia e proatividade.

Atualmente, estou no último semestre da faculdade de Análise e Desenvolvimento de Sistemas. Durante o curso, desenvolvi projetos como um jogo da memória em C# .NET, um jogo de cartas de 21 utilizando C# .NET e um site de freelance de empregos em grupo, onde usei PHP, HTML, CSS, MySQL e JavaScript. Para o desenvolvimento dos projetos em C#, utilizei o Visual Studio como ambiente de desenvolvimento integrado (IDE), o que me permitiu aproveitar as funcionalidades avançadas dessa ferramenta para otimizar a codificação e depuração.

Atualmente, estou participando do bootcamp da Generation Brasil, onde me especializo em desenvolvimento fullstack Java. Tem me permitido aprimorar minha lógica de programação e desenvolver projetos utilizando Java e Spring Boot. Um dos projetos mais significativos foi o desenvolvimento de uma API de blog pessoal, onde utilizei o padrão MVC, implementei JWT Security e trabalhei com React no frontend, criando uma interface interativa com HTML, CSS e JS. Durante o bootcamp, também aprimorei meus conhecimentos em arquitetura de software, integração com bancos de dados, criação de API REST, boas práticas de desenvolvimento, testes automatizados e controle de versionamento com GIT.
